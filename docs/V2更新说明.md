# V2 更新说明

## 新增能力

1. 规则档案保存与加载。
2. 工作簿结构签名与匹配校验。
3. 目标预设随规则档案保存。
4. 目标预设逐条校验并标红。
5. UI 页签结构修正。
6. 新增年份写回时优先使用“来源行 + 年份列”。
7. 方法注释按“是什么 / 为什么”补充。
8. 新增中文使用说明和开发文档。

## 业务口径

- 最新现状年默认 2025；
- 同时率范围 0.83-0.99，调整不超过 0.1；
- 容载比默认硬边界 1.3-2.5；
- 配变平均负载率 0-1，软目标 0.5；
- 区外送受电为低优先级兜底；
- 项目投产年不可修改；
- 原始上传文件不修改，导出副本写回调整值。

## 已知限制

1. 区县分摊目前按现有负荷占比。
2. 规则项编辑仍以目标预设为主，尚未做到所有指标单元格级编辑。
3. 区外送受电站点仍需人工确认。
4. 容量侧项目联动尚未完整自动化。
