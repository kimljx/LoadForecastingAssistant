# 负荷预测与容载比反推助手 - 使用说明

## 1. 软件用途

本工具面向供需预测及平衡分析类 Excel 模板，用于辅助完成：

- 根据上级指定的容载比目标反推未来负荷；
- 根据增长率目标生成平滑预测值；
- 校核容载比、配变平均负载率、同时率等关键指标；
- 在导出的 Excel 副本中写入调整值；
- 自动生成“预测结果表”，解释目标达成情况、调整项和写回清单。

> 原始传入 Excel 永远不会被修改。软件只会在用户确认导出时生成一个新 Excel 文件。

---

## 2. 支持文件

支持：

- `.xlsx`
- `.xls`

导出统一为：

- `.xlsx`

`.xls` 文件会先读取并转换为内部临时结构，最终仍导出为 `.xlsx`。

## 3. 快速开始

### 3.1 启动软件

开发环境运行：

```bash
python main.py
```

打包后运行：

```text
双击 LoadForecastingAssistant.exe
```

### 3.2 打开模板

点击：

```text
打开Excel模板
```

选择供需预测模板文件。

打开后软件会自动完成：

1. 工作簿读取；
2. Sheet 角色识别；
3. 动态年份识别；
4. 关键指标抽取；
5. 规则档案自动加载；
6. 兼容性检测。

---

## 4. 模板识别

在“模板识别”页，可以查看每个 Sheet 的识别结果。

常见角色包括：

- 地市容载比表；
- 区县容载比表；
- 校核表；
- 直供负荷明细；
- 变电容量明细；
- 电源装机明细；
- 储能装机明细；
- 110/35kV 项目库。

项目库不依赖固定名称。以下名称均可识别：

```text
十四五 十五五 110-35明细表
十五五 十六五 110-35明细表
十六五 十七五 110-35明细表
110-35项目明细表
110-35规划项目表
```

软件主要通过字段识别项目库，例如：

```text
项目名称、所在区县、电压等级、增加变电容量、开工年月、投产年月、投产年
```

## 5. 目标设置

进入“目标设置”页，可设置一个或多个反推目标。

支持目标类型：

| 目标类型 | 说明 |
|---|---|
| 容载比 | 根据容量和目标容载比反推网供负荷 |
| 增长率 | 根据阶段增长率生成平滑负荷预测 |
| 负荷 | 根据指定年份负荷目标生成调整方案 |
| 配变平均负载率 | 按 10kV 负荷 / 变电容量需求校核 |

### 5.1 容载比目标示例

```text
区域：内江市
电压等级：110kV
目标类型：容载比
目标年份：2030
目标值：1.85
```

含义：

```text
让 2030 年内江市 110kV 容载比接近 1.85。
```

### 5.2 增长率目标示例

```text
区域：内江市
电压等级：110kV
目标类型：增长率
目标年份/阶段结束年：2030
目标下限：5%
目标上限：7%
```

含义：

```text
控制 2025–2030 年 110kV 网供负荷复合增长率在 5%–7%。
```

### 5.3 多目标

点击：

```text
添加目标到列表
```

可以建立多目标任务，例如同时控制：

```text
2030年 110kV 容载比
2030年 35kV 容载比
2025-2030年 增长率
2035年 110kV 容载比
```

目标列表为空时，系统使用当前表单中的单个目标。

## 6. 规则管理

进入“规则管理”页，可以查看和修改当前业务规则。

默认规则：

| 规则 | 默认值 |
|---|---|
| 最新现状年 | 2025 |
| 容载比范围 | 1.3–2.5 |
| 配变负载率范围 | 0–1 |
| 配变负载率软目标 | 0.5 |
| 同时率范围 | 0.83–0.99 |
| 同时率最大调整 | 0.1 |
| 单年增长率范围 | -2%–12% |
| 区外送受电 | 允许，低优先级兜底 |

### 6.1 保存规则档案

点击：

```text
保存规则档案
```

软件会默认保存为：

```text
rules/{模板主名称}_规则.yaml
```

例如：

```text
供需预测及平衡分析模板_规则.yaml
```

规则档案包括：

- 当前规则边界；
- 目标预设；
- 工作簿结构签名；
- 来源模板名称。

### 6.2 自动加载规则

下次打开同主名称模板时，软件会自动查找并加载对应规则。

若模板结构变化，软件会提示：

```text
规则可能不完全匹配
```

若规则引用的目标预设在当前模板中找不到，会标为红色。

### 6.3 删除失效目标预设

点击：

```text
删除失效目标预设
```

软件会删除当前目标列表中状态为：

```text
missing
invalid
```

的目标预设。

这不会删除整个规则文件，只会清理不适用于当前模板的目标项。

## 7. 生成反推方案

设置目标和规则后，点击：

```text
生成反推方案
```

软件会生成三类方案：

| 方案 | 说明 |
|---|---|
| 优先调整负荷 | 主要通过未来负荷增长率和网供负荷调整满足目标 |
| 负荷容量均衡 | 负荷和容量侧共同承担调整压力 |
| 区外送受电兜底 | 正常调整不足时，提示新增区外送（+）受（-）电 |

推荐优先查看：

```text
优先调整负荷
```

因为这是最符合日常负荷预测反推工作的方案。

- 区域；
- 电压等级；
- 指标；

展示：

- 方案说明；
- 风险提示；
- 区外送受电是否被使用；
- 同时率是否超出范围。

## 9. 导出新 Excel

点击：

```text
导出新Excel
```

软件会执行：

1. 复制原始工作簿；
2. 在副本中写入预测年份调整值；
3. 必要时新增 2031、2032、2035 等年份列；
4. 新增 `预测结果表`；
5. 保存为新文件。

导出文件命名示例：

```text
供需预测及平衡分析模板_反推结果_20260617_103000.xlsx
```

### 9.1 关于公式单元格

界面中有选项：

```text
导出时允许覆盖公式单元格
```

如果勾选，导出副本中会用结果值覆盖对应公式单元格。

如果不勾选，遇到公式单元格会跳过写入，并在“预测结果表”的写回清单中记录原因。

正式生产建议：

```text
优先写入底层输入项，让 WPS 打开后由模板公式自动重算。
```

当前版本仍保留覆盖公式的选择，是为了满足人工反推结果快速落表的需求。

---

## 10. 注意事项

1. 2025 年及以前默认为现状数据，不允许修改。
2. 项目投产年不可修改。
3. 区外送受电为低优先级兜底变量，使用时必须人工确认站点和口径。
4. 同时率允许小幅调整，但范围为 0.83–0.99，单次调整不超过 0.1。
5. 新增年份列会复制相邻年份列样式，但不会保证所有复杂公式完全自动扩展。
6. 导出前请检查“风险提示”和“写回单元格清单”。

## 11. 常见问题

### Q1：软件会修改我的原始 Excel 吗？

不会。软件只会在导出时复制原文件，然后修改副本。

### Q2：为什么项目库 sheet 名变了还能识别？

软件主要通过字段识别项目库，而不是固定 sheet 名。

### Q3：2031 年原模板没有列，能预测吗？

可以。软件内部支持新增年份，导出副本时会在相关 sheet 中新增年份列，并在“预测结果表”中记录。

### Q4：规则为什么会标红？

说明规则中的区域、电压等级、指标或目标在当前模板中无法匹配。可以删除失效目标预设，或重新设置目标。

### Q5：容载比目标达不到怎么办？

优先检查：

1. 增长率上下限是否过窄；
2. 容量是否固定；
3. 同时率是否已经到边界；
4. 是否允许使用区外送受电兜底。
